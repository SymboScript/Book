# Literals

In SymboScript exists only three kinds of token literals: strings, numbers, booleans.

## Strings

Strings are enclosed in double quotes, single quotes or backticks.

```ts
"hello";
"hello"`hello`;
("hello\nworld");
('hello"world');
("hello ' world");
```

## Numbers

```ts
1 .1
1.1
1e1
1E1
1.1e1
1.1E1
```

## Booleans

```ts
true;
false;
```
